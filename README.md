# 🛠️ Mini JavaScript Projects

This repository contains a collection of small, beginner-to-intermediate level JavaScript projects built using HTML, CSS, and JavaScript. These projects are designed to improve core JS skills and demonstrate functionality using DOM manipulation, event handling, and basic logic implementation.

---

## 🌟 Projects Included

### 1. 💸 Tip Calculator
A simple calculator that allows users to input a bill amount and tip percentage to calculate the total tip and grand total.

**Features:**
- User input for bill amount and tip percentage
- Real-time calculation
- User-friendly UI

---

### 2. 🎂 Age Calculator
Calculate your exact age based on the date of birth entered.

**Features:**
- Input date of birth
- Calculates years, months, and days
- Handles leap years and future dates gracefully

---

### 3. ⏱️ Stopwatch
A digital stopwatch that allows start, stop, and reset functionalities.

**Features:**
- Millisecond precision
- Start/Stop toggle
- Reset button

---

### 4. ✨ Quote Generator
Displays random inspirational quotes on button click.

**Features:**
- Random quote logic
- Clean UI with each quote displayed prominently
- Easy to add more quotes

---

### 5. 💱 Currency Breakdown
Breaks down a given amount into currency denominations (notes/coins).

**Features:**
- Input any amount
- Returns optimal note breakdown (100s, 50s, 10s, etc.)
- Simple and interactive layout

---

## 🔧 Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
